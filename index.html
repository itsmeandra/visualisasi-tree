Visualisasi Game Pohon Biner

Bangun Binary Search Tree secara visual dengan gaya retro.

Cara Bermain

  1. Masukkan Angka: Ketik angka pada kotak input.
  2. Tambahkan Node: Klik tombol "Tambahkan Node".
  3. Aturan Pohon: Angka kecil ke kiri, angka besar ke kanan.
  4. Reset Pohon: Klik "Reset Pohon" untuk mulai dari awal.

    5. Kegunaan Tree: Menemukan informasi spesifik dengan sangat cepat, Contohnya: Mencari nama teman di kontak ponsel di antara ratusan nama lainnya.
